# Figma-to-React Project

A modern React frontend built from Figma designs using **Tailwind CSS** and popular UI libraries.


### Tech Stack & Libraries
- **React 19**  
- **Tailwind CSS 4**   
- **React Icons**   
- **Lucide React** 
- **Recharts** 
- **RC Progress** 
- **Swiper**  
- **Marquee**

### Installation
```bash
git clone https://github.com/Sinthia-22/PotentialAI_Task.git
cd figma-to-react
npm install
npm run dev
```
